Customer Service KPI Analysis (2025)

**Project Overview: **
This project is an end-to-end data analytics portfolio project focused on analyzing customer service key performance indicators (KPIs) for the year 2025. The objective is to evaluate service efficiency and customer experience by comparing baseline targets versus actual performance, and to present insights through an interactive Power BI dashboard.

The project demonstrates practical use of SQL, Python, and Power BI in a real-world analytics workflow.

**Objectives:**
* Calculate baseline and actual customer service KPIs using SQL
* Clean and prepare raw data using Python
* Analyze monthly performance trends and deviations from baseline
* Visualize insights in a clear and business-friendly Power BI dashboard
* Support data-driven decision-making for customer service improvement

**Key Metrics Analyzed:**
* First Response Time (Minutes)
* First Resolution Time (Minutes)
* Full Resolution Time (Minutes)
* Customer Satisfaction Rate (CSAT) (%)
* First Contact Resolution Rate (FCR) (%)

**Tools & Technologies**
* SQL – KPI calculations, baseline vs actual analysis
* Python – Data cleaning, transformation, and validation
* Power BI – Data modeling and interactive dashboard visualization

**Power Bi Dashboard Highlights**
* Clear KPI cards showing current values and variance vs baseline
* Monthly trend analysis for each metric
* Baseline reference lines for quick performance comparison
* Interactive slicers for year and timeframe selection

**Disclaimer**
This project is for educational and portfolio purposes only. The data used is either simulated or anonymized and does not represent any real organization.
